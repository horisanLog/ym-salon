# Y'm 整体サロン Web サイト 要件定義書

**作成日：2025 年 11 月 9 日**  
**バージョン：1.0**  
**ステータス：初版ドラフト**

---

## 1. 目的（Purpose）

### サイトの実現目標

- **プレオープン期間中に月間 80 件の予約獲得**（初期目標）
- AI 姿勢診断×整体×エステの総合サロンとしてのブランド確立
- 地域密着型サロンとして戸田市周辺での認知度向上
- プレオープン特別価格による新規顧客の獲得と継続利用促進

### 競合サロンとの差別化ポイント

- **AI 姿勢診断ツールによる科学的アプローチ**：データに基づいた施術提案
- **整体×エステ×電磁パルスの総合メニュー**：1 つのサロンで多様なケアが可能
- **プレオープン特別価格**：体験のハードルを下げる戦略的価格設定
- **個別カスタマイズ施術**：AI 診断結果に基づく一人ひとりに最適化されたプラン

### プレオープン戦略

- **期間限定特別価格**：通常価格から 30〜50% OFF の体験価格設定
- **緊急性の訴求**：「先着○名様限定」「○月○日まで」の限定性表現
- **口コミ・紹介促進**：初期顧客からの紹介制度で認知拡大

## 2. ターゲット（Persona）

### 主要ペルソナ

| 属性         | 詳細                                                      |
| ------------ | --------------------------------------------------------- |
| **年齢層**   | 25〜55 歳（特に 30〜45 歳の働く女性）                     |
| **職業**     | 会社員（デスクワーク中心）、主婦、自営業                  |
| **居住地域** | 戸田市、蕨市、さいたま市南部、板橋区北部                  |
| **年収**     | 300〜700 万円                                             |
| **利用経験** | 整体・エステ経験者、姿勢や体の不調に悩みを持つ            |

### 抱えている体の悩み・課題

- **デスクワークによる肩こり・腰痛・姿勢の悪化**
- 慢性的な疲労感・頭痛
- 産後の骨盤の歪み・体型の変化
- 美容面での悩み（顔のたるみ・むくみ・くびれ不足）
- 運動不足による筋力低下・血行不良

### 来訪前の心理状態／求めている価値

- 整体やエステに通いたいが、効果が実感できるか不安
- 複数のサロンに通うのは時間的・金銭的に負担
- 科学的根拠のある施術を受けたい
- プレオープン価格なら試してみたい
- 個人の状態に合わせたカスタマイズ施術を求める

### サロン利用経験・予算感

- **利用経験**：整体・マッサージ経験者が 70%、エステ経験者が 50%
- **予算感**：1 回あたり 3,000〜8,000 円、月 1〜2 回利用
- **重視点**：効果実感 > 価格 > 立地・アクセス > 雰囲気

## 3. コンバージョン目標（Goal）

### 一次 CV（LINE 予約・電話予約）

- **目標 CVR**：4〜6%（サイト訪問者ベース）
- **月間目標**：80 件（プレオープン期間）
- **予約完了後の来店率**：85% 以上
- **予約方法別比率**：LINE 予約 60%、電話予約 30%、フォーム予約 10%

### 二次 CV（LINE 友だち登録／Instagram フォロー／資料請求）

- **LINE 公式アカウント友だち登録**：月間 100 件
- **Instagram フォロー**：月間 50 件
- **AI 姿勢診断の詳細資料請求**：月間 30 件
- **ブログ記事の定期閲覧**：リピート訪問促進

### 予約率・来店率・リピート率の KPI（仮値）

| 指標                        | 目標値        | 備考           |
| --------------------------- | ------------- | -------------- |
| サイト訪問者数              | 月間 1,500 人 | SEO・SNS 流入  |
| LP→ 予約フォーム遷移率      | 20%           | CTA の効果測定 |
| 予約フォーム完了率          | 70%           | EFO 対策必須   |
| 初回来店 → リピート転換率   | 60%           | 施術品質重要   |

## 5. 非機能要件（Non-functional）

### パフォーマンス

- **LCP（Largest Contentful Paint）**：2.5 秒以内（特にモバイル重視）
- **FID（First Input Delay）**：100ms 以内
- **CLS（Cumulative Layout Shift）**：0.1 以下
- **モバイルページ速度スコア**：85 以上（PageSpeed Insights）

### SEO 対策

- **ローカル SEO 対策**：Google マイビジネス連携、地域名の自然配置
- **title/meta description 最適化**：各ページ固有設定
- **構造化データ対応**：LocalBusiness、Service、Review
- **XML サイトマップ自動生成**
- **OGP 設定**：SNS シェア対応

### 保守性

- **CMS で更新可能**：メニュー・料金・お客様の声・ブログ記事
- **画像アップロード・リサイズ自動化**
- **キャンペーン情報の簡単更新機能**
- **非エンジニアでも更新可能な管理画面**

### セキュリティ

- **reCAPTCHA v3 実装**：フォーム保護
- **CSRF 対策**：トークン検証
- **SSL/TLS 暗号化**：必須
- **個人情報の暗号化保存**
- **写真使用許可管理**

### アクセシビリティ

- **WCAG 2.1 AA 準拠を目標**
- **キーボード操作対応**
- **スクリーンリーダー対応**：aria-label、alt 属性
- **色のコントラスト比 4.5:1 以上**
- **フォントサイズ調整可能**

## 7. リスク・留意点

### 法規制・コンプライアンス

- **医療法・薬機法**：治療効果表現の規制（「治る」「治療」等の表現は避ける）
- **景品表示法**：ビフォーアフター写真使用時の個人差明記
- **個人情報保護法**：写真使用許可、プライバシーポリシー整備
- **特定商取引法**：料金表示、キャンセル規定明記

### 技術・運用リスク

- **LINE 予約システムの連携不具合対策**
- **画像・動画ファイルサイズによるページ速度低下**
- **CMS 更新権限管理の設計**
- **AI 姿勢診断機器の技術的説明の正確性担保**

### ブランド・評判リスク

- **お客様の声の真実性担保**（虚偽は厳禁）
- **施術効果の個人差に関する適切な説明**
- **SNS 炎上対策**（適切な投稿ガイドライン）
- **競合他社との差別化ポイントの明確化**

## 9. デザイン・ブランディング方針

### カラーパレット

- **メインカラー**：ホワイト（#FFFFFF）- 清潔感・信頼感
- **サブカラー**：ベージュ（#F5F5DC）- 温かみ・安心感
- **アクセントカラー**：薄グリーン（#E8F5E8）- 自然・癒し・AI 感
- **テキストカラー**：ダークグレー（#333333）- 可読性重視

### フォント

- **メインフォント**：Noto Sans JP（安心感・読みやすさ）
- **アクセントフォント**：Roboto（近未来感・AI 技術感）

### 写真表現

- **「データ × リラックス」を意識**
- AI 診断機器や施術風景を自然光で撮影
- 清潔感のあるサロン内観
- お客様の笑顔・リラックスした表情
- ビフォーアフターの比較画像（許可取得済み）

### CTA 配置

- **各セクション末に「LINE 予約はこちら」を配置**
- **フローティング予約ボタン**（スマホ画面下部固定）
- **ヘッダーに電話番号と LINE 予約ボタン**

## 10. マーケティング戦略

### プレオープン特別価格の訴求方法

- **期間限定性の強調**：「○月○日まで」「先着○名様限定」
- **価格比較の明示**：通常価格との差額を明確に表示
- **体験ハードル軽減**：「まずはお試し」「お気軽に」の文言使用

### ローカル SEO 対策（地域名＋サービス名）

- **ターゲットキーワード**：「戸田市 整体」「蕨 AI 姿勢診断」「さいたま市南部 エステ」
- **Google マイビジネス最適化**：営業時間・写真・口コミ管理
- **地域密着コンテンツ**：近隣情報・アクセス詳細

### SNS 運用方針（Instagram 中心、施術風景・ビフォーアフター）

- **Instagram**：施術風景・サロン内観・スタッフの日常
- **投稿頻度**：週 3〜4 回
- **ハッシュタグ戦略**：地域名＋サービス名の組み合わせ

### 口コミ・紹介制度の設計

- **紹介特典**：紹介者・被紹介者双方に割引適用
- **口コミ投稿促進**：Google レビュー・Instagram 投稿で次回割引
- **満足度調査**：施術後のアンケート実施
