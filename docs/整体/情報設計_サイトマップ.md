## 1. 情報構造マップ（Information Architecture）

サイト全体を 3〜4 階層で整理し、「知る」「選ぶ」「予約する」「信頼を得る」「来店する」など、ユーザー意図ごとに分類

```
├── ホーム（LP）
│   ├── サロンの特徴・差別化ポイント
│   ├── AI 姿勢診断紹介
│   ├── メニュー・料金概要
│   ├── お客様の声・ビフォーアフター
│   ├── 予約 CTA（LINE・電話）
│   └── アクセス・営業時間
├── メニュー・料金詳細 (/menu)
│   ├── 整体コース (/menu/seitai)
│   ├── エステコース (/menu/esthe)
│   ├── 電磁パルス (/menu/pulse)
│   └── お得メニュー (/menu/campaign)
├── AI 姿勢診断について (/ai-diagnosis)
├── お客様の声・実績 (/voice)
├── スタッフ・サロン紹介 (/about)
├── 予約・お問い合わせ (/reservation)
├── アクセス・店舗情報 (/access)
├── ブログ・コラム (/blog)
└── 法務・企業情報 (/privacy, /terms, /company)
```

## 4. 導線設計（User Flow）

### メイン導線
1️⃣ トップ → サロンの特徴・AI 姿勢診断を知る  
2️⃣ → メニュー・料金を確認  
3️⃣ → お客様の声・実績を確認  
4️⃣ → 予約（LINE・電話・フォーム） → 予約完了  

### 補助導線
- ブログ・コラム → 専門性訴求 → 予約誘導  
- アクセス情報 → 来店イメージ → 予約誘導  
- スタッフ紹介 → 安心感 → 予約誘導  
- 予約完了後 → LINE 友だち登録 or Instagram フォロー

## 6. 地域密着型サロン特有の設計

### ローカル SEO 対策
- 各ページに地域名（戸田市、蕨市、さいたま市南部）を自然に配置
- Google マイビジネス連携（営業時間・口コミ・写真）
- 構造化データ（LocalBusiness）の実装

### 来店促進要素
- 最寄り駅からの詳細な道順（写真付き）
- 駐車場情報・台数・料金
- 近隣ランドマーク情報
- 初回来店時の流れ・所要時間

### 地域コミュニティ連携
- 地域イベント参加情報
- 近隣店舗との連携・紹介制度
- 地域密着型のブログコンテンツ

## 7. 予約システム連携設計

### LINE 予約システム
- LINE Bot による自動予約受付
- 希望日時・コース選択・お客様情報入力
- 予約確認・リマインド機能
- キャンセル・変更対応

### 電話予約
- ヘッダー・フッター・固定バーに電話番号配置
- 営業時間外の自動音声案内
- 電話タップ計測（GA4 イベント）

### フォーム予約（補助）
- LINE が使えない方向けの Web フォーム
- 必要最小限の項目設計
- 確認画面・完了画面の設計

## 10. 詳細ページ構成

### ホーム（LP）- `/`

**目的**：認知→関心→信頼→予約の導線構築  
**セクション構成**：
1. ヒーロー（AI 姿勢診断×整体×エステの訴求）
2. サロンの 3 つの特徴（差別化ポイント）
3. プレオープン特別価格（緊急性・限定性）
4. メニュー・料金概要（主要 4 コース）
5. AI 姿勢診断紹介（科学的根拠・プロセス）
6. お客様の声（ビフォーアフター・満足度）
7. スタッフ・サロン紹介（安心感・信頼性）
8. アクセス・営業時間（来店準備）
9. FAQ（よくある質問・不安解消）
10. 最終予約 CTA（LINE・電話・フォーム）

### メニュー・料金 - `/menu`

**目的**：サービス内容の詳細理解と予約誘導  
**セクション構成**：
1. メニュー概要（整体・エステ・電磁パルスの特徴）
2. カテゴリタブ（整体／エステ／電磁パルス／キャンペーン）
3. 整体コース詳細（4 コース・料金・所要時間・効果）
4. エステコース詳細（3 コース・美容効果・ビフォーアフター）
5. 電磁パルス詳細（機器説明・効果・科学的根拠）
6. お得メニュー（セット料金・プレオープン価格）
7. 施術の流れ（来店〜施術〜アフターケア）
8. 予約・問い合わせ CTA

### AI 姿勢診断 - `/ai-diagnosis`

**目的**：差別化ポイントの詳細説明と体験予約誘導  
**セクション構成**：
1. AI 姿勢診断概要（他サロンとの違い）
2. 診断の仕組み・科学的根拠
3. 診断プロセス（3D 撮影〜分析〜結果説明）
4. ビフォーアフター事例（許可取得済み）
5. 改善プラン例（個別カスタマイズ）
6. 体験予約 CTA

### お客様の声 - `/voice`

**目的**：信頼醸成と共感による予約誘導  
**セクション構成**：
1. お客様の声概要（満足度・改善実績）
2. カテゴリ絞り込み（肩こり・腰痛・姿勢改善・美容効果）
3. 体験談カードリスト（写真・年齢・職業・コメント・改善内容）
4. 統計・満足度データ（数値での信頼性訴求）
5. ビフォーアフター写真ギャラリー
6. 予約導線（「あなたも体験してみませんか？」）

### スタッフ・サロン紹介 - `/about`

**目的**：安心感・信頼性向上と予約誘導  
**セクション構成**：
1. サロンの理念・コンセプト
2. スタッフプロフィール（経歴・保有資格・専門分野）
3. 保有資格・経験年数（信頼性の根拠）
4. サロン内観・設備紹介（清潔感・最新機器）
5. 安心への取り組み（衛生管理・プライバシー保護）
6. 予約・問い合わせ CTA

### 予約・お問い合わせ - `/reservation`

**目的**：予約完了への最終導線  
**セクション構成**：
1. 予約方法選択（LINE・電話・フォーム）
2. LINE 予約手順（QR コード・友だち登録・予約方法）
3. 電話予約案内（営業時間・番号・注意事項）
4. フォーム予約（希望日時・コース・お客様情報）
5. 予約時の注意事項（キャンセル規定・持ち物）
6. 初回来店の流れ（受付〜カウンセリング〜施術）

### アクセス・店舗情報 - `/access`

**目的**：来店準備と予約誘導  
**セクション構成**：
1. アクセス概要（最寄り駅・所要時間）
2. Google マップ埋め込み（現在地からのルート）
3. 最寄り駅からの詳細道順（写真付きガイド）
4. 駐車場情報（台数・料金・近隣パーキング）
5. 営業時間・定休日・祝日対応
6. 近隣ランドマーク・目印情報
7. 予約・問い合わせ CTA

### ブログ・コラム - `/blog`

**目的**：SEO 対策・専門性訴求・リピート促進  
**セクション構成**：
1. ブログ概要（更新頻度・カテゴリ紹介）
2. カテゴリ・タグ絞り込み（姿勢改善・美容・セルフケア・サロンニュース）
3. 記事カードリスト（サムネイル・タイトル・概要・投稿日）
4. 人気記事ランキング
5. 予約導線（「詳しくはサロンで相談」）
