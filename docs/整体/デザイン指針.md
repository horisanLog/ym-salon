# Y'm 整体サロン デザイン指針

**作成日：2025 年 11 月 9 日**  
**バージョン：1.1**  
**ステータス：更新版（ロゴ適用）**

---

## 1. デザインコンセプト

- **テーマ**：清潔感 × 癒し × 科学的根拠 + プレミアム感（王冠モチーフ）
- **印象**：安心・信頼・リラックス・最新技術・上質感
- **トーン**：ホワイトベース／ベージュ・薄グリーンアクセント＋クラウンロゴに合わせたゴールド
- **目的**：「データ × リラックス」を表現しつつ、ロゴが持つ王冠（Y'm Crown）の象徴性をサイト全体に展開する
- **キービジュアル要素**：王冠シルエット／ゴールドの光沢感／柔らかなボケ感を持つ写真背景

## 2. カラーパレット

| 用途         | カラー             | コード例 | 備考                                 |
| ------------ | ------------------ | -------- | ------------------------------------ |
| 背景         | ホワイト           | #FFFFFF  | 全体ベース・清潔感                   |
| メイン       | ベージュ           | #F5F5DC  | 温かみ・安心感                       |
| アクセント A | 薄グリーン         | #E8F5E8  | 自然・癒し・AI 技術感                |
| アクセント B | クラウンゴールド   | #C7A253  | ロゴのゴールドと統一。強調箇所に使用 |
| テキスト     | ダークグレー       | #333333  | 可読性重視                           |
| サブテキスト | ミディアムグレー   | #6B7280  | 補足説明                             |
| 成功・特典   | ソフトグリーン     | #22C55E  | プレオープン価格・効果訴求           |
| 注意・緊急性 | ソフトオレンジ     | #F59E0B  | 期間限定・予約満席時                 |
| 仕上げライン | シャンパンホワイト | #FDF8EE  | セクション境界や背景パターンに使用   |

### カラールール

- ゴールド（#C7A253）はボタンの縁取りやタイトル下ライン、アイコン背景など「品質・プレミアム」を伝える箇所に限定して使用し、ベタ塗り面積は 20% 以下に抑える。
- ゴールドと薄グリーンの併用時はゴールドを前面（線・グラデーションのハイライト側）に置き、薄グリーンで柔らかさを補う。
- ロゴ周辺の背景はホワイト or ベージュを基本とし、カラーノイズや過度なグラデーションを避ける。

### フォント運用

- ロゴと合わせる特別な箇所（ヒーローのサブタイトルやキャンペーン見出し）は Playfair Display をアクセント的に使用し、文字色はゴールドもしくはダークグレー。
- 通常本文は Noto Sans JP をベースに統一し、日本語テキストの可読性を重視する。
- 数値／診断結果などは Roboto を使用し、科技感を演出。

## 4. レイアウトルール

### SP ファースト

- **max-width**: 640px
- **セクション間**: `py-8`（上下 32px）
- **内側余白**: `px-4`（左右 16px）
- **予約 CTA バー固定**（下部）: `fixed bottom-0 left-0 w-full z-50`
- **タップ領域**: 最小 44px × 44px

### PC

- **コンテナ**: `max-w-screen-xl mx-auto px-8`
- **2 カラムレイアウト**: 画像左・テキスト右の交互配置
- **セクション背景交互**（白 ⇄ 薄ベージュ）で視認性 UP
- **ホバーエフェクト**: `hover:shadow-lg transition-shadow duration-300`

## 5. 主要コンポーネント設計

### 🦸 Hero（更新）

- ロゴを左上に配置し、余白上下 24px・左右 32px 以上確保。
- キャッチコピー下にゴールドの 2px 下線（長さ 80px）を追加。
- CTA ボタンは「LINE 予約（グリーン）」「電話（アウトライングリーン）」「予約ページ（ゴールド縁取り）」の 3 種類を基本とする。
- 背景は王冠の曲線を想起させるソフトな波形グラデーション（ゴールド → ホワイト → 薄グリーン）。

### 🏥 サロン特徴カード（更新）

- アイコン背景はゴールド or 薄グリーンのいずれかで統一。
- カード上部に王冠の稜線をイメージした 3px ゴールドラインを追加可能。

### 💰 メニュー比較表（更新）

- 人気メニューのバッジ背景はゴールド、文字をホワイトに変更。
- 価格差を表現する際、通常価格 → ゴールド（線）、特別価格 → グリーン（面）で視認性を高める。

### 🤖 AI 姿勢診断紹介

- **構成**: 診断プロセス図解／ビフォーアフター写真／科学的根拠
- **プロセス**: `flex items-center space-x-4` でステップ表示
- **ビフォーアフター**: `grid grid-cols-2 gap-4` で比較表示
- **体験ボタン**: `bg-blue-500 text-white rounded-lg py-3 px-6 font-semibold hover:bg-blue-600`

### 💬 お客様の声スライダー

- **構成**: 写真／年齢・職業／コメント／星評価
- **スライダー**: `overflow-x-auto flex space-x-4 pb-4`
- **カード**: `min-w-80 bg-white rounded-xl shadow-md p-6 flex-shrink-0`
- **星評価**: `text-yellow-400` で ★★★★★ 表示
- **写真**: `rounded-full w-16 h-16 object-cover`

### 👩‍⚕️ スタッフ・サロン紹介

- **レイアウト**: 写真左・プロフィール右（SP は縦積み）
- **写真**: `rounded-lg shadow-md w-full md:w-48 h-64 object-cover`
- **資格バッジ**: `inline-block bg-green-100 text-green-800 px-3 py-1 rounded-full text-sm mr-2 mb-2`
- **サロン内観**: `grid grid-cols-2 md:grid-cols-3 gap-4` でギャラリー表示

### 🗺️ アクセスマップ

- **Google マップ**: `w-full h-64 md:h-80 rounded-lg shadow-md`
- **駐車場情報**: `bg-blue-50 border-l-4 border-blue-500 p-4 rounded`
- **最寄り駅**: `flex items-center space-x-2` でアイコン + テキスト
- **営業時間**: `grid grid-cols-2 gap-2 text-sm`

### 📅 予約システム

- **予約方法選択**: `grid grid-cols-1 md:grid-cols-3 gap-4`
- **LINE 予約**: `bg-green-500 text-white` + LINE アイコン

- **QR コード**: `mx-auto w-32 h-32`

### 🚀 予約 CTA バー（SP 固定）

- **表示条件**: 常時表示（重要性が高いため）
- **スタイル**: `fixed bottom-0 left-0 w-full bg-white border-t shadow-lg z-50`
- **ボタン配置**: `grid grid-cols-2 gap-2 p-3`
- **LINE**: `bg-green-500 text-white rounded-lg py-3 font-semibold`
- **電話**: `bg-blue-500 text-white rounded-lg py-3 font-semibold`

## 6. ロゴ運用ガイドライン（新規）

1. **最小表示サイズ**：横幅 64px 以上。縮小時は王冠内部の線が潰れないよう留意。
2. **余白規定**：ロゴの上下左右に「Y」の高さと同等の余白を確保。
3. **背景色**：ホワイトまたはベージュで使用。写真背景に載せる場合は王冠部分に 70% 白のオーバーレイを敷く。
4. **禁止事項**：色変更・縦横比の変形・ドロップシャドウ・縁取り追加は不可。王冠と「Y'm」を分離しない。
5. **ファビコン**：王冠のみを正円に収めたバリエーション（ゴールド #C7A253 + ホワイト）を使用。
